# Grupo5-CofreEletronico
Grupo 5 - Cofre Eletrônico

O presente projeto tem como objetivo o desenvolvimento de um cofre eletrônico com tranca magnética, controlado por um microcontrolador STM32F103C8T6, que garante segurança, praticidade e confiabilidade no armazenamento de objetos de valor. O sistema combina elementos de eletrônica digital, controle e automação, aplicando conceitos de programação embarcada, interface homem-máquina e acionamento de cargas por relé.

O funcionamento do cofre é baseado em um teclado matricial 4x4, utilizado para a inserção de uma senha numérica. O microcontrolador interpreta as teclas pressionadas, valida o código digitado e, caso a senha esteja correta, aciona a tranca magnética por meio de um relé controlado por um transistor MOSFET. O sistema também inclui LEDs indicadores (verde e vermelho) e um buzzer, que fornecem retorno sonoro e visual ao usuário sobre o estado do sistema — seja de acesso liberado, erro de senha ou tentativa de invasão.

A segurança lógica é reforçada pela implementação de travamento após três tentativas incorretas consecutivas e pela presença de um watchdog interno, que evita travamentos do programa. O cofre ainda conta com armazenamento não volátil da senha em memória Flash/EEPROM, o que permite manter os dados mesmo após desligamento da energia. O circuito foi projetado para operar com alimentação de 5V, possuindo regulador de 3,3V para o microcontrolador, além de proteções elétricas como fusível e diodo de flyback na bobina do relé. Também foi prevista a possibilidade de alimentação auxiliar por bateria, garantindo o funcionamento mesmo em caso de queda de energia.

A tranca magnética empregada é capaz de suportar forças de retenção acima de 60 kgf, assegurando resistência física adequada. Um reed switch é utilizado para detectar o estado da porta (aberta ou fechada), permitindo que o sistema só acione a tranca quando o cofre estiver devidamente fechado. Essa integração entre sensores, atuadores e controle lógico garante não apenas a funcionalidade esperada, mas também um alto nível de segurança e confiabilidade operacional.

O firmware desenvolvido é modular, separando funções de leitura de teclado, validação de senha, controle de saída e gerenciamento de estados. A estrutura do código facilita futuras atualizações, como comunicação serial (UART) para depuração ou integração com sistemas IoT. Além disso, o projeto foi pensado para ser didático e expandível, podendo ser utilizado em ambientes acadêmicos ou como base para protótipos comerciais de sistemas de segurança.

Em síntese, o cofre eletrônico com tranca magnética controlado por STM32 combina hardware robusto, software confiável e interface intuitiva, demonstrando uma aplicação prática dos princípios de sistemas embarcados e controle eletrônico. O projeto alia segurança, baixo custo e eficiência energética, destacando-se como uma solução moderna e versátil para proteção de bens pessoais e corporativos.
